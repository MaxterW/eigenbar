# eigenbar

Eine digitale Getränkekarte für meine private Homebar, erreichbar über einen QR-Code.

Die Anwendung ersetzt eine klassische PDF-Karte durch eine interaktive Webseite, die auf Smartphones optimiert ist. Gäste können Cocktails durchsuchen, Informationen zu den Drinks ansehen und ihre Bewertungen sowie Kommentare hinterlassen.
